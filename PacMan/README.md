# Neon Maze

A modern neon arcade maze-chase game made with Python and Pygame. It uses procedural, high-DPI-friendly graphics, so there are no image assets to download.

## Run

```powershell
cd "Neon Maze"
python main.py
```

If Pygame is not installed:

```powershell
pip install -r requirements.txt
```

## Controls

- Arrow keys or WASD — move
- P — pause
- R — restart
- Enter or Space — start / play again
- Esc — quit

## Features

- Neon-glow maze and modern HUD
- Four ghosts with distinct chase behaviour
- Power cores, ghost combo scoring, tunnel wrapping, fruit bonus, lives and levels
- No external art files: sprites, UI and particle effects render dynamically
